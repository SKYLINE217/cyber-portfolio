# 🌐 Cyber-Forensics Portfolio

A high-performance, visually immersive personal portfolio website built with a "Cyber-Forensics Terminal" aesthetic. This project showcases my professional cybersecurity background, project history, and technical skill set through a dynamic and interactive web application.

## 🚀 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **Scrolling:** [Lenis](https://lenis.studiofreight.com/) (Antigravity momentum scrolling)

## ✨ Features

- **Cyber-Terminal UI:** A unique, dark-mode focused aesthetic inspired by cybersecurity forensics tools.
- **Fluid Animations:** Smooth page transitions, scroll-triggered animations, and micro-interactions powered by Framer Motion and GSAP.
- **Performance Optimized:** Highly optimized asset delivery, efficient rendering, and custom scroll physics.
- **Security Hardened:** Implements secure headers, XSS protections, and strict content security policies.
- **Responsive Design:** Fully functional and visually striking across desktop, tablet, and mobile devices.

## 💻 Featured Projects

Highlights from the portfolio include:
- **WinVolAuto Memory Forensics:** Automated memory analysis and forensics tool.
- **Blockchain BATA:** Blockchain-based architecture for trust and authentication.
- **Real-Time Fraud Detection:** High-speed data pipeline for detecting anomalous and fraudulent activities.

## 🛠️ Getting Started

First, clone the repository and install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🔒 Security & Performance

This project underwent a professional-grade security and performance audit:

- Fine-tuned Lenis scroll for optimal mobile/desktop reliability.
